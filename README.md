#Bahasa-Lo REPL FINAL v5

Ultimate Bahasa Indonesia REPL dengan fitur lengkap: admin, plugin, proot-distro, file explorer, backup, dan semua command Python dalam Bahasa Indonesia.

## Cara Install

1. Pastikan Python 3 terpasang:
```bash
python3 --version ```
2. Pastikan package penting sudah terinstall:
```Bash
apt update
apt install -y git wget curl rsync proot-distro```
3. Download Script:
```Bash
git clone https://github.com/m29752885-glitch/Bahasa-lo.git
cd Bahasa-lo```
4. Jalankan REPL:
```Bash
python bahasa-lo.py```
